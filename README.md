#Calculadora Pro en Python

Aplicación de calculadora con interfaz gráfica desarrollada en Python usando Tkinter.

---

##Características

* Interfaz gráfica moderna (modo oscuro)
* Operaciones básicas (+, -, *, /)
* Botón de limpiar (C)
* Diseño responsivo
* Manejo de errores

---

##Tecnologías

* Python
* Tkinter

---

##Cómo ejecutar

```bash
python calculadora.py
```
